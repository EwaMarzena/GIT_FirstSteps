How repository is built?

  1. Repository
  2. Staging area
  3. Working directory: tracked, untracked

    4. Working directory -> (git add)-> Staging area -> git commit -m"MESSAGE"-> Repository

  • git init - initalization of repository
  • ls - show files in actual folder
  • ls -a - show hidden folders
  • git status - repository status

    • COMMIT is a change made in repository

  • touch 1.txt - creat file 1.txt
  • code . - open actual catalog usin VS
  • git add 1.txt - track 1.txt
  • git commit -m"MESSAGE" - commiting file with description
  • git log - history of commits
  • git add -A - adding all the files to staging area
